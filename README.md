# TACSFON Bookshop and Stationeries

An e-commerce web app for TACSFON's church bookshop, allowing students to browse and order stationeries online with Paystack payment integration.

## Tech Stack
- **Frontend:** Next.js (Vercel)
- **Database & Auth:** Supabase
- **Payments:** Paystack

## Features
- Browse stationery products by category
- Cart & checkout with delivery address
- Paystack payment integration
- Order tracking
- Staff dashboard for inventory & order management
